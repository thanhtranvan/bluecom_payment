# bluecom_payment
Install and using extension:
	1. copy -> paste extension to your "app" folder.
	2. Go to admin page: 
		2.1 At the menu "system -> configuration -> sales -> payment method -> bluecom payment (stripe)".
		2.2 Set the  API key is "sk_test_mgpJA8eNKFOSxtMECnRJTTmE" (this is my API key to test - you can get it for yourself in page "stripe.com").
		2.3 Choose the "Credit Card Types" and click button "save config".
	3. Go to the Front end and test the result.
